# Examples for RouterOS API

This folder contains several examples to help you get started with the `routeros_api` package.

## 1. Quick Start
**File:** `routeros_api_example.dart`
The basic "Hello World" of this package. Shows how to connect and list network interfaces.

## 2. Live Bandwidth Monitor
**File:** `live_monitor.dart`
Demonstrates real-time data streaming using the `listen()` method. Watch your interface traffic live.

## 3. Network Dashboard
**File:** `network_dashboard.dart`
Advanced usage showing how to correlate data from different RouterOS modules (DHCP and Hotspot) to create a meaningful insight dashboard.

## 4. Robust Error Handling
**File:** `error_handling_guide.dart`
Essential for production apps. Learn how to catch authentication failures, command traps, and operation timeouts properly.
